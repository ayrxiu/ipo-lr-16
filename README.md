Веб-приложение на Django про рыболовный магазин.

1.Главная страница(`/`) — приветствие и ссылки на разделы.
2.Об авторе(`/about/`) — информация о студенте, выполнившем работу.
3.О магазине(`/info/`) — описание тематики магазина (товары для рыбалки).

Склонировать: git clone https://github.com/ayrxiu/ipo-lr-16
Создать окружение: python -m venv venv.
Активировать: venv\Scripts\activate.
Установить Django: pip install django.
Запустить: python manage.py runserver.
